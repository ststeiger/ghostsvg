Incompatible changes

  • The file gs/lib/CIDFnmap used in Ghostscript 6.53 and 7.0x to map CIDFont resource is no longer supported. Now it is replaced with gs/lib/cidfmap. TrueType font substition to CIDFont is not implemented yet.
