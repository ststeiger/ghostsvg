Version 6.20 (2000-xx-xx)


This fileset includes major additions and revisions to bring the pdfwrite
driver closer to matching Acrobat Distiller 4.  This also requires upgrading
several aspects of the library and the PostScript and PDF interpreters.


Incompatible changes


(Drivers)
	- The default BufferSpace for printer drivers is now 4 Mb.
(Streams)
	- The interface to the [T]BCP filters is now in a separate file,
sbcp.h.
(Library)
	- ImageType 4 images now give an error if a MaskColor value is out
of the range of possible sample values.
