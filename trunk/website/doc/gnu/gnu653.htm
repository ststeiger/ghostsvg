Release of GNU Ghostscript 6.53

We are pleased to announce the release of GNU Ghostscript 6.53. This release is an important security update. See doc/News.htm for details.

Also new in this release:

  • autoconf build support
  • improved asian language support
  • integrates the IJS driver
  • integrates the Gimp-Print driver
  • the usual misc bugfixes

Sourceforge and cs.wisc.edu are the primary distribution sites for this release.

Thanks to everybody who helped on this release.
