Release of GNU Ghostscript 6.52

We are pleased to announce the release of GNU Ghostscript 6.52. This is a maintenance release containing all of the bug fixes committed to the stable branch since the 6.51 release. In particular, the build issues with glibc 2.2 on linux are resolved.
