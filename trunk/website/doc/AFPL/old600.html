Obtaining Aladdin Ghostscript 6.0

See also the topic Ghostscript Licensing.

Post-release news

After you get this version of Ghostscript, we suggest that you follow this link to find a list of problems, fixes, and workarounds for this release.

Mirror sites

The primary FTP site for Ghostscript is ftp://ftp.cs.wisc.edu/ghost. Ghostscript is also available from many mirror sites which may be closer to you and provide faster access. Ghostscript is also available on CD-R.

Unix & VMS

You will need to obtain the following sources:
  • ghostscript-6.0.tar.gz, Ghostscript sources, 3450043 bytes
  • jpegsrc.v6b.tar.gz, JPEG library sources, 613261 bytes
  • libpng-1.0.3.tar.gz, LIBPNG library sources, 232899 bytes
  • zlib-1.1.3.tar.gz, ZLIB library sources, 168463 bytes
and also the following fonts:
  • ghostscript-fonts-std-6.0.tar.gz, Ghostscript standard fonts, 1881034 bytes
Start by reading the Readme.htm file in ghostscript-6.0.tar.gz.

It is also worthwhile having the GV or Ghostview previewer.

Precompiled RPM files are available from ghostscript-6.0-1.i386.rpm and ghostscript-fonts-6.0-1.noarch.rpm, The sources are available from ghostscript-6.0-1.src.rpm and ghostscript-fonts-6.0-1.src.rpm. These are known to work on the following Linux variants:

  • RedHat Linux 5.1 (the RPMs were built on this platform).
  • RedHat Linux 6.0.
  • Caldera OpenLinux 2.2.
  • Caldera OpenLinux 1.3 provided you first install the runtime glibc package from the /col/contrib/RPMS directory on the OpenLinux 1.3 "Installation" CD-ROM.

OS/2, MS-DOS & MS-Windows

These files might not be available until about two weeks after the release of Aladdin Ghostscript 6.0.

Windows 95, 98 or NT

You will need to obtain and run the self-extracting EXE
gs600w32.exe, Aladdin Ghostscript 6.0 for Win32 4821504 bytes
Alternatively, a plain zip file is available.

It is recommended that you also install GSview, a graphical interface to Ghostscript.

gsv28w32.exe, GSview 2.8 for Win32 ~1M bytes

Windows 3.1

To run the 32-bit executables, you will need to obtain and install Win32s 1.30, download the self extracting EXE mentioned in the previous section, unzip it using a tool such as Info-ZIP, then manually install it using the instructions in Install.htm. The self extracting archive will not unpack itself correctly under Win32s. The enclosed setup program will not run under Win32s.

If you need 16-bit exectuables, you will need to obtain and run the self-extracting EXE

gsv27403.exe, GSview 2.7 and Aladdin Ghostscript 4.03 for Win16 2936095 bytes
This is the most recent 16-bit version.

OS/2

You will need to obtain the following ZIP files:
  • gs600os2.zip, Aladdin Ghostscript for OS/2, ~5M
  • gsv28os2.zip, GSview for OS/2, ~1M,
Unzip gsv28os2.zip and follow the instructions in the file Readme.htm.

DOS

There is no DOS EXE prebuilt for Aladdin Ghostscript 6.0. See Aladdin Ghostscript 5.10 for a 32-bit DOS EXE. Alternatively, the Win32 package above includes gswin32c.exe, a Win32 command line version of Ghostscript.

Compiling from sources

If you plan to re-compile from the sources, you will need gs600w32.exe or gs600os2.zip plus:
  • gs600src.zip, Ghostscript sources, 3049k
  • jpegsr6b.zip, Independent JPEG group sources, 745k
  • lpng103.zip, PNG graphics library, 260k
  • zlib113.zip, zlib compression library, 214k
The documentation for Ghostscript is typically installed in the directory c:/Aladdin/gs6.0/doc. Look for Readme.htm, New-user.htm, Install.htm and Use.htm.

GSview

If using OS/2 or MS-Windows it is strongly recommended that you also get GSview, which provides a graphical interface to Ghostscript.

To unpack ZIP files, you will need an unzip program such as Info-ZIP unzip.

Macintosh

Aladdin Ghostscript 6.0 will be available precompiled for the Mac here.

The following information applies to Aladdin Ghostscript 5.50.

You will need to obtain one of the following:

  • macgs-550-68k.sit.bin, application for 68020 or better machines, 573k
  • macgs-550-ppc.sit.bin, application for PowerPC version, 734k
  • macgs-550-fat.sit.bin, application for both PowerPC and 68k platforms, 1.26M
plus all of these:
  • README.txt, installation instructions, 2k
  • macgs-550-files.sit.bin, configuration, initialization, and example files, 883k
  • macgs-550-fonts.sit.bin, full set of Ghostscript's Type 1 fonts, 1.48M
If you plan to re-compile from the sources, you will need:
  • ghostscript-5.50.tar.gz, Ghostscript sources 2826251 bytes
  • jpegsrc.v6b.tar.gz, JPEG library sources, 613261 bytes
  • libpng-1.0.2.tar.gz, LIBPNG library sources, 225386 bytes
  • zlib-1.1.3.tar.gz, ZLIB library sources, 168463 bytes
  • macgs-550-src.sit.bin, additional sources for the Macintosh, 227k
Start by reading the readme.txt file for installation instructions.

The MacGS Viewer Manual is part of the macgs-550-files.sit.bin archive and contains more detailed information. There is also a web version of the manual on-line. You can find out more about MacGS-550 on-line as well.
