AFPL Ghostscript 8.14

Artifex Software, Inc. and artofcode LLC are pleased to announce the release of AFPL Ghostscript 8.14. This is the fourth release in the stable 8.1x series.

This release fixes a common problem view some files with antialiased output; we recommend that all users upgrade to this version. There is also a new: the PDF writer now implements PDF 1.4 encryption. See News.htm in the source distribution for details.

The release is available from: sourceforge.net, mirror.cs.wisc.edu and their mirrors.

As usual, unix source tarballs are available now; builds and source archive for other platforms will follow in a few days.
