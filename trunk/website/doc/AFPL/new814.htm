AFPL Ghostscript 8.14

Posted 22 Feb 2004 by rillian

Artifex Software, Inc. and artofcode LLC are pleased to announce the release of AFPL Ghostscript 8.14. This releases fixes a common issue with anti-aliased rendering and upgrading is recommended. Also new in this release is support for encrypted PDF output.

This is the fourth release in the stable 8.1x series and it includes a number of minor bugfixes as well. See the changelog in the source distribution for details.
