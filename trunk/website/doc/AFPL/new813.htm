AFPL Ghostscript 8.13

Artifex Software, Inc and artofcode LLC are pleased to announce the availability of Ghostscript 8.13.

This is the third stable release in the 8.1x series. It fixes some build issues and a crashing bug with the ps2epsi but is otherwise identical to 8.12.

Over the 8.11 release, it offers numerous bug fixes; other improvements include improved font rendering and colorspace handling, and unicode text support for high-level devices.

AFPL Ghostscript 8.13 is available from mirror.cs.wisc.edu and sourceforge.net. and their mirrors.

Please see Obtaining AFPL Ghostscript 8.13 for more details.
