Obtaining AFPL Ghostscript 7.03

See also the topic Ghostscript Licensing.

Post-release news

After you get this version of Ghostscript, we suggest that you track news, updates, and fixes on SourceForge. Please visit http://sourceforge.net/projects/ghostscript/.

You can also participate in the discussion at the Ghostscript community website, www.ghostscript.com.

Mirror sites

The primary site for the Ghostscript source code is http://download.sourceforge.net/ghostscript. The primary FTP site for the full Ghostscript release is ftp://mirror.cs.wisc.edu/pub/mirrors/ghost. Ghostscript is also available from many mirror sites which may be closer to you and provide faster access. Ghostscript is also available on CD-R.

Unix & VMS

You will need to obtain the following sources:
  • ghostscript-7.03.tar.gz, Ghostscript sources, 4669597 bytes
  • jpegsrc.v6b.tar.gz, JPEG library sources, 613261 bytes
  • libpng-1.0.12.tar.gz, LIBPNG library sources, 481387 bytes
  • zlib-1.1.3.tar.gz, ZLIB library sources, 168463 bytes
and also the following fonts:
  • ghostscript-fonts-std-6.0.tar.gz, Ghostscript standard fonts, 1881034 bytes
Start by reading the Readme.htm file in ghostscript-7.03.tar.gz.

It is also worthwhile having the GV or GSview previewer.

Precompiled RPM files will soon be available.

Windows 95, 98, NT or 2000

You will need to obtain and run the self-extracting EXE

gs703w32.exe, AFPL Ghostscript 7.03 for Win32 6187008 bytes
You can unzip this if you want to check the contents.

It is recommended that you also install GSview, a graphical interface to Ghostscript. GSview 3.6 will not work with AFPL Ghostscript 7.03. You will need to use GSview 4.0.

gsv40w32.exe, GSview 4.0 for Win32 1491456 bytes

OS/2

You will need to obtain the following ZIP files:

  • gs703os2.zip, AFPL Ghostscript for OS/2, 5961514 bytes
  • gsv40os2.zip, GSview for OS/2, 1190887 bytes,
Unzip gsv40os2.zip and follow the instructions in the file Readme.htm.

DOS

There is no DOS EXE prebuilt for AFPL Ghostscript 7.03. See Aladdin Ghostscript 5.10 for a 32-bit DOS EXE. Alternatively, the Win32 package above includes gswin32c.exe, a Win32 command line version of Ghostscript.

Compiling from sources

If you plan to re-compile from the sources, you will need gs703w32.exe or gs703os2.zip plus:
  • gs703src.zip, Ghostscript sources, 4663k
  • jpegsr6b.zip, Independent JPEG group sources, 745k
  • lpng1012.zip, PNG graphics library, 556k
  • zlib113.zip, zlib compression library, 214k
The documentation for Ghostscript is typically installed in the directory c:/gs/gs7.03/doc. Look for Readme.htm, New-user.htm, Install.htm and Use.htm.

GSview

If using OS/2 or MS-Windows it is strongly recommended that you also get GSview, which provides a graphical interface to Ghostscript.

To unpack ZIP files, you will need an unzip program such as Info-ZIP unzip.

Macintosh

AFPL Ghostscript 7.03 will be available precompiled for the Mac here.

The following information applies to Aladdin Ghostscript 5.50.

You will need to obtain the following: macgs-550-installer.bin, FAT binary, configuration, initialization, fonts, and example files, 4551k

Note: This download requires StuffitExpander version 5.x or any version of MindExpander for decompression. Older versions of StuffitExpander will complain that the archive is corrupted; this is not the case.

If you plan to re-compile from the sources, you will need:

  • ghostscript-5.50.tar.gz, Ghostscript sources 2826251 bytes
  • jpegsrc.v6b.tar.gz, JPEG library sources, 613261 bytes
  • libpng-1.0.2.tar.gz, LIBPNG library sources, 225386 bytes
  • zlib-1.1.3.tar.gz, ZLIB library sources, 168463 bytes
  • macgs-550-src.sit.bin, additional sources for the Macintosh, 227k
Start by reading the readme.txt file for installation instructions.

The MacGS Viewer Manual is part of the macgs-550-installer.bin archive and contains more detailed information. There is also a web version of the manual on-line. You can find out more about MacGS-550 on-line as well.
