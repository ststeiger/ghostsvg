Obtaining Aladdin Ghostscript 5.10

See also the topic Ghostscript Licensing.

Post-release news

After you get this version of Ghostscript, we suggest that you follow this link to find a list of problems, fixes, and workarounds for this release.

Mirror sites

The primary FTP site for Ghostscript is ftp://mirror.cs.wisc.edu/pub/mirrors/ghost. Ghostscript is also available from many mirror sites which may be closer to you and provide faster access. Ghostscript is also available on CD-R.

Unix & VMS

You will need to obtain the following sources:
  • ghostscript-5.10.tar.gz, Ghostscript sources, 2595853 bytes
  • jpegsrc.v6a.tar.gz, JPEG library sources, 538910 bytes
  • libpng-0.96.tar.gz, LIBPNG library sources, 152722 bytes
  • zlib-1.0.4.tar.gz, ZLIB library sources, 86288 bytes
and also the following fonts:
  • ghostscript-fonts-std-5.10.tar.gz, Ghostscript standard fonts, 1444863 bytes
Start by reading the README file in ghostscript-5.10.tar.gz.

It is also worthwhile having the GV previewer.

A precompiled version for RedHat Linux 5.1 is available from ghostscript-5.10-1.i386.rpm and ghostscript-fonts-5.10-2.noarch.rpm, The sources are available from ghostscript-5.10-1.src.rpm and ghostscript-fonts-5.10-2.src.rpm.

OS/2, MS-DOS & MS-Windows

You will need to obtain the following ZIP files:

  • gs510ini.zip, configuration, initialization, and example files, 805k
Plus one or more of:
  • gs510dos.zip, MS-DOS 32-bit, 890k
  • gs510os2.zip, OS/2 32-bit, 727k
  • gs510w32.zip, MS-Windows 32-bit (Win32s, Win95, WinNT), 556k
  • gs510wda.zip, MS-Windows 32-bit for the DEC Alpha (WinNT), 1071k. This was not compiled by Aladdin and cannot be tested by Aladdin.
If you don't already have a full set of Type 1 fonts, you will also need:
  • gs510fn1.zip, Ghostscript standard fonts, 1143k
If you plan to re-compile from the sources, you will need:
  • gs510sr1.zip, Ghostscript sources part 1/3, 1236k
  • gs510sr2.zip, Ghostscript sources part 2/3, 1311k
  • gs510sr3.zip, Ghostscript sources part 3/3, 698k
  • jpeg-6a.zip, Independent JPEG group sources, 657k
  • lpng096.zip, PNG graphics library, 166k
  • zlib104.zip, zlib compression library, 106k
The font metrics (AFM files) for the fonts in gs510fn1.zip and some extra fonts are available in:
  • gs510fn2.zip, Ghostscript font metrics and extra fonts, 1053k.
The documentation for Ghostscript is in the gs510ini.zip file. Look for README, new-user.txt, install.txt and use.txt.

If using OS/2 or MS-Windows it is recommended that you also get GSview, which provides a graphical interface to Ghostscript. GSview 2.5 contains an installation program which also installs Ghostscript 5.10. You need ftp://mirror.cs.wisc.edu/pub/mirrors/ghost/rjl/old/gsview25.zip 2336k, gs510ini.zip, gs510fn1.zip and either gs510w32.zip or gs510os2.zip. See the README.TXT file in gsview25.zip for more details. The latest version available for the DEC Alpha is GSview 2.4.

To unpack ZIP files, you will need an unzip program such as Info-ZIP unzip.

Nelson Beebe has some notes on installing GSview and Ghostscript on Windows 95.

Macintosh

Aladdin Ghostscript 5.10 is available precompiled for the Mac here.

The following information applies to Aladdin Ghostscript 5.10.

You will need to obtain one of the following:

  • macgs-510-68k.sit.bin, application for 68020 or better machines, 573k
  • macgs-510-ppc.sit.bin, application for PowerPC version, 734k
  • macgs-510-fat.sit.bin, application for both PowerPC and 68k platforms, 1.26M
plus all of these:
  • README.txt, installation instructions, 2k
  • macgs-510-files.sit.bin, configuration, initialization, and example files, 883k
  • macgs-510-fonts.sit.bin, full set of Ghostscript's Type 1 fonts, 1.48M
If you plan to re-compile from the sources, you will need:
  • ghostscript-5.10.tar.gz, Ghostscript sources
  • jpegsrc.v6a.tar.gz, JPEG library sources, 538910 bytes
  • libpng-0.96.tar.gz, LIBPNG library sources, 152722 bytes
  • zlib-1.0.4.tar.gz, ZLIB library sources, 86288 bytes
  • macgs-510-src.sit.bin, additional sources for the Macintosh, 227k
Start by reading the readme.txt file for installation instructions.

The MacGS Viewer Manual is part of the macgs-510-files.sit.bin archive and contains more detailed information. There is also a web version of the manual on-line. You can find out more about MacGS-510 on-line as well.
