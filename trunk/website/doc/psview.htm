ps_view previewer

ps_view provides a set of GSview-like commands that MS-DOS users can enter through the keyboard. This package, created by GUST, the Polish TeX Users Group, is in the public domain.

It is available from 

	ftp://ftp.pg.gda.pl/pub/TeX/support/ps_view/*
and from sites in the CTAN network such as 
	ftp://ftp.dante.de/tex-archive/support/ps_view/*

The authors describe the package as follows:

Fast and easy previewing of PS documents!
a PostScript preamble providing an interactive environement for previewing PostScript documents. The current version can drive the current version of the Ghostscript interpreter; enables scaling, rotating, restart, on-line help, printing the screen, etc. Batch files provided for MS DOS, however ps_view can be adapted to any operating system.

Authors: Boguslaw Jackowski and Piotr Pianowski
For more information, e-mail to <P.Pianowski@gust.org.pl>.
