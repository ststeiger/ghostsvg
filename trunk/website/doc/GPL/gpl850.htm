Release of GPL Ghostscript 8.50

artofcode, LLC. and Artifex Software, Inc. are pleased to announce the release of GPL Ghostscript 8.50. In addition to many bug fixes it contains numerous new features. Most obvious will be improved font handling and rasterization, and better support for PDF 1.4 and 1.5 documents.

This program was originally released 2004-12-10 as AFPL Ghostscript 8.50. This version is the same code released under the GNU GPL license.

Source archives and binaries are available from the usual locations and mirrors:

http://sourceforge.net/projects/ghostscript/
ftp://mirror.cs.wisc.edu/pub/mirrors/ghost/GPL/gs850/
