MacGSView Preview for Mac OS X

Preview Warning!

This is a preview version of MacGSView for Mac OS X. This version has not been tested very much and there are still many problems. Use it at your own risk!

A fully carbonized and tested version will be released in a few weeks.

Obtaining MacGSView Preview for Mac OS X

You can download the Mac OS X Preview of MacGSView as StuffIt archive:

  • MacGSView Preview for Mac OS X (5.2 MB)

The archive contains the carbonized Ghostscript 6.01 software as well as a carbonized version of MacGSView.

Installation Instructions

  1. Download and unstuff the StuffIt archive
  2. Put the MacGSView folder to an appropriate location like /Applications. (Note: this version does not need any files to be installed outside the MacGSView folder)
  3. Run the MacGSView application

Known Issues

  • Printing is not implemented
  • Crashes from time to time, but thanks to Mac OS X, this is no longer a problem for other applications
  • Glitches in the user interface
  • ....
