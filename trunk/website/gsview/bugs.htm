(2001-06-10)

Problem: (4.0) DOS EPS file is invalid if adding TIFF preview and EPS omits trailer. If an EPS file omits the trailer and a TIFF preview is added by GSview or epstool, the DOS EPS file will contain an incorrect size for the PostScript section. The same bug occurs if a user supplied preview is added. This does not affect adding a Windows MetaFile preview.

Workaround: Append the following line to the EPS file before adding the preview 

 %%Trailer
