GSview BETA

The latest release version is GSview 3.6.

GSview 3.61 beta test is now available. This includes Windows and OS/2 executables. The major changes are:

  • Uses new Ghostscript interpreter API (expect bugs)
  • Fixes a few bugs as documented in the GSview Readme.htm

GSview 3.61 beta can be found in ftp://mirror.cs.wisc.edu/pub/mirrors/ghost/ghostgum/beta/

Linux

This is not yet available. With some minor changes GSview 3.61 would probably compile for Linux. In a month or so I hope to have a another test release which will use the new Ghostscript shared object under Linux.

Win32

GSview 3.61 beta for Win32 is available from ftp://mirror.cs.wisc.edu/pub/mirrors/ghost/ghostgum/beta/gsv361w32.exe.

You will also need AFPL Ghostscript 7.0.

OS/2

GSview 3.61 beta for OS/2 is available from ftp://mirror.cs.wisc.edu/pub/mirrors/ghost/ghostgum/beta/gsv361os2.zip

You will also need AFPL Ghostscript 7.0.

General

This beta will expire at the end of June 2001
