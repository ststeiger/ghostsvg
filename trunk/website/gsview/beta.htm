GSview BETA

The latest release version is GSview 4.4.

GSview 4.41 beta test is now available. This includes Windows, OS/2 and Linux executables. The major changes are:

  • Fix Page Offset when printing.
  • Fix slow behaviour when displaying on remote X11 server.
  • Fix crash in X11 version when the LANG environment variable is not set.
  • A few other minor bug fixes.

GSview 4.41 beta can be found in ftp://mirror.cs.wisc.edu/pub/mirrors/ghost/ghostgum/beta/

Linux

Sources are available at ftp://mirror.cs.wisc.edu/pub/mirrors/ghost/ghostgum/beta/gsview-4.41beta.tar.gz.

A tar file containing a precompiled gsview executable is available in ftp://mirror.cs.wisc.edu/pub/mirrors/ghost/ghostgum/beta/gsview-4.41beta.i386.tar.gz.

You will also need GNU Ghostscript 7.05 or AFPL Ghostscript 8.11 . In particular, you need to install both the ghostscript executable and shared library.

Win32

GSview 4.41 beta for Win32 is available from ftp://mirror.cs.wisc.edu/pub/mirrors/ghost/ghostgum/beta/gsv441w32.exe.

You will also need Ghostscript 7.04, 8.11 or later.

OS/2

GSview 4.41 beta for OS/2 is available from ftp://mirror.cs.wisc.edu/pub/mirrors/ghost/ghostgum/beta/gsv441os2.zip

You will also need Ghostscript 7.04, 8.11 or later.

General

Sources are in ftp://mirror.cs.wisc.edu/pub/mirrors/ghost/ghostgum/beta/gsv441src.zip.

This beta will expire at the end of November 2003.
