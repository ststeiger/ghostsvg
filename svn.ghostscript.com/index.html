Source repositories

The following repositories are available at this site:

  • ghostscript Our venerable PS/PDF parse/render/convert engine
  • jasper Our modified version of the JasPer JPEG 2000 library
  • jbig2dec OBSOLETE The current tree is here for our JBIG2 decoder library.

For example, you can check out the current Ghostscript development tree like so:

svn co http://svn.ghostscript.com/ghostscript/trunk/gs

Or the current GhostPDL development tree:

svn co http://svn.ghostscript.com/ghostscript/trunk/ghostpdl

There is also an svn mirror at code.google.com which has a browse interface.

Or you can use the local viewvc access to browse the history of both the CVS and SVN repositories. Just select a different Project root to access the different SVN trees.

Repositories and mirrors for some trees are also available for git, darcs, and mercurial.
