Post-release news for Aladdin Ghostscript 6.01 has moved

We no longer post bug information or fixes here: we are now using the bug tracking system on SourceForge, where we have also moved our development source code. For more information, please visit http://sourceforge.net/project/?group_id=1897.

We are aware that the SourceForge bug tracking system has some serious shortcomings:

  • It doesn't provide a standard (selectable or searchable) way to identify in what release a bug was reported or fixed.
  • It doesn't support classifying bugs by platform.
  • It doesn't provide a convenient way to link from a closed bug report to the changes that fix the bug.
  • It doesn't provide any search facilities.

We are planning to work with SourceForge to upgrade their system to be more useful, in these ways among others.
