AFPL Ghostscript 7.03

artofcode LLC and Artifex Software Inc. are pleased to announce the release of AFPL Ghostscript version 7.03. This release is a maintenance release in the 7.0x branch. Numerous bug fixes and performance enhancements are included. We heartily recommend that you upgrade.

We hope that this will be the last release in the 7.0x branch. With this release, the CVS tree becomes a development branch again.

Source tarballs and Windows executables and, for the first time in the 7.x series, a MacOS shlib build are available from the SourceForge and wisc.edu download sites.

Thanks to everybody who helped on this release.

Raph Levien
2001-10-21
