Obtaining Aladdin Ghostscript 5.03

See also the topic Ghostscript Licensing.

Mirror sites

The primary FTP site for Ghostscript is ftp://ftp.cs.wisc.edu/ghost. Ghostscript is also available from many mirror sites which may be closer to you and provide faster access. Ghostscript is also available on CD-R.

Unix & VMS

You will need to obtain the following sources:
  • ghostscript-5.03.tar.gz, Ghostscript sources, 2553690 bytes
  • jpegsrc.v6a.tar.gz, JPEG library sources, 538910 bytes
  • libpng-0.96.tar.gz, LIBPNG library sources, 152722 bytes
  • zlib-1.0.4.tar.gz, ZLIB library sources, 86288 bytes
and also the following fonts:
  • ghostscript-fonts-std-5.03.tar.gz, Ghostscript standard fonts, 1442181 bytes
If you already have the sources for Ghostscript 5.02, then instead of obtaining the 5.03 sources, you may wish to patch the 5.02 sources using:
  • ghostscript-5.02-5.03.diff.gz, Ghostscript 5.02 to 5.03 source diffs, 10025 bytes
Start by reading the README file in ghostscript-5.03.tar.gz.

It is also worthwhile having the Ghostview previewer.

OS/2, MS-DOS & MS-Windows

You will need to obtain the following ZIP files:
  • gs503ini.zip, configuration, initialization, and example files, 794k
Plus one or more of:
  • gs503dos.zip, MS-DOS 32-bit, 845k
  • gs503os2.zip, OS/2 32-bit, 692k
  • gs503w32.zip, MS-Windows 32-bit (Win32s, Win95, WinNT), 546k
If you don't already have a full set of Type 1 fonts, you will also need:
  • gs503fn1.zip, Ghostscript standard fonts, 1143k
If you plan to re-compile from the sources, you will need:
  • gs503sr1.zip, Ghostscript sources part 1/3, 1223k
  • gs503sr2.zip, Ghostscript sources part 2/3, 1288k
  • gs503sr3.zip, Ghostscript sources part 3/3, 684k
  • jpeg-6a.zip, Independent JPEG group sources, 657k
  • lpng096.zip, PNG graphics library, 166k
  • zlib104.zip, zlib compression library, 106k
The documentation for Ghostscript is in the gs503ini.zip file. Look for README, new-user.txt, install.txt and use.txt.

If using OS/2 or MS-Windows it is recommended that you also get GSview, which provides a graphical interface to Ghostscript. GSview 2.3 contains an installation program which also installs Ghostscript 5.03. You need gsview23.zip (2233k), gs503ini.zip, gs503fn1.zip and either gs503w32.zip or gs503os2.zip. See the README.TXT file in gsview23.zip for more details.

Macintosh

Aladdin Ghostscript 5.03 is now available precompiled for the Mac.

The following information applies to Aladdin Ghostscript 5.03.

You will need to obtain one of the following:

  • MacGS-503-68k.sit.bin, application for 68020 or better machines, 573k
  • MacGS-503-PPC.sit.bin, application for PowerPC version, 734k
  • MacGS-503-FAT.sit.bin, application for both PowerPC and 68k platforms, 1.26M
plus all of these:
  • README.txt, installation instructions, 2k
  • MacGS-503-FILES.sit.bin, configuration, initialization, and example files, 883k
  • MacGS-503-FONTS.sit.bin, full set of Ghostscript's Type 1 fonts, 1.48M
If you plan to re-compile from the sources, you will need:
  • ghostscript-5.03.tar.gz, Ghostscript sources
  • jpegsrc.v6a.tar.gz, JPEG library sources, 538910 bytes
  • libpng-0.96.tar.gz, LIBPNG library sources, 152722 bytes
  • zlib-1.0.4.tar.gz, ZLIB library sources, 86288 bytes
  • MacGS-503-src.sit.bin, additional sources for the Macintosh, 227k
Start by reading the readme.txt file for installation instructions.

The MacGS Viewer Manual is part of the MacGS-503-FILES.sit.bin archive and contains more detailed information. There is also a web version of the manual on-line. You can find out more about MacGS-503 on-line as well.
