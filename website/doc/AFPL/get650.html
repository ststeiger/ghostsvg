Obtaining AFPL Ghostscript 6.50

See also the topic Ghostscript Licensing.

Post-release news

After you get this version of Ghostscript, we suggest that you track news, updates, and fixes on SourceForge. Please visit http://sourceforge.net/project/?group_id=1897.

You can also participate in the discussion at the Ghostscript community website, www.ghostscript.com.

Mirror sites

The primary site for the Ghostscript source code is http://download.sourceforge.net/ghostscript. The primary FTP site for the full Ghostscript release is ftp://ftp.cs.wisc.edu/ghost. Ghostscript is also available from many mirror sites which may be closer to you and provide faster access. Ghostscript is also available on CD-R.

Unix & VMS

You will need to obtain the following sources:
  • ghostscript-6.50.tar.gz, Ghostscript sources, 4240377 bytes
  • jpegsrc.v6b.tar.gz, JPEG library sources, 613261 bytes
  • libpng-1.0.8.tar.gz, LIBPNG library sources, 484520 bytes
  • zlib-1.1.3.tar.gz, ZLIB library sources, 168463 bytes
and also the following fonts:
  • ghostscript-fonts-std-6.0.tar.gz, Ghostscript standard fonts, 1881034 bytes
Start by reading the Readme.htm file in ghostscript-6.50.tar.gz.

It is also worthwhile having the GV or Ghostview previewer.

Precompiled RPM files are available from ghostscript-6.50-1.i386.rpm and ghostscript-fonts-6.0-2.noarch.rpm, The sources are available from ghostscript-6.50-1.src.rpm and ghostscript-fonts-6.0-2.src.rpm. These are expected to work on the following Linux variants:

  • RedHat Linux 6.2 (the RPMs were built on this platform).
  • Caldera OpenLinux 2.2.
  • Caldera OpenLinux 1.3 provided you first install the runtime glibc package from the /col/contrib/RPMS directory on the OpenLinux 1.3 "Installation" CD-ROM.

Please note that these RPM files were built with a limited set of printer drivers. If you need extra printer drivers, you will need to recompile Ghostscript.

Windows 95, 98 or NT

You will need to obtain and run the self-extracting EXE

gs650w32.exe, Aladdin Ghostscript 6.50 for Win32 5877760 bytes
Alternatively, a plain zip file is available.

It is recommended that you also install GSview, a graphical interface to Ghostscript. Please try GSview 3.42 beta which fixes a compatibility problem with AFPL Ghostscript.

gsv34w32.exe, GSview 3.4 for Win32 1180672 bytes

OS/2

You will need to obtain the following ZIP files:

  • gs650os2.zip, Aladdin Ghostscript for OS/2, 5672756 bytes
  • gsv34os2.zip, GSview for OS/2, 899754 bytes,
Unzip gsv34os2.zip and follow the instructions in the file Readme.htm.

DOS

There is no DOS EXE prebuilt for AFPL Ghostscript 6.50. See Aladdin Ghostscript 5.10 for a 32-bit DOS EXE. Alternatively, the Win32 package above includes gswin32c.exe, a Win32 command line version of Ghostscript.

Compiling from sources

If you plan to re-compile from the sources, you will need gs650w32.exe or gs650os2.zip plus:
  • gs650src.zip, Ghostscript sources, 3288k Note: this file is not yet available
  • jpegsr6b.zip, Independent JPEG group sources, 745k
  • lpng108.zip, PNG graphics library, 541k
  • zlib113.zip, zlib compression library, 214k
The documentation for Ghostscript is typically installed in the directory c:/gs/gs6.50/doc. Look for Readme.htm, New-user.htm, Install.htm and Use.htm.

GSview

If using OS/2 or MS-Windows it is strongly recommended that you also get GSview, which provides a graphical interface to Ghostscript.

To unpack ZIP files, you will need an unzip program such as Info-ZIP unzip.

Macintosh

Aladdin Ghostscript 6.50 will be available precompiled for the Mac here.

The following information applies to Aladdin Ghostscript 5.50.

You will need to obtain the following: macgs-550-installer.bin, FAT binary, configuration, initialization, fonts, and example files, 4551k

Note: This download requires StuffitExpander version 5.x or any version of MindExpander for decompression. Older versions of StuffitExpander will complain that the archive is corrupted; this is not the case.

If you plan to re-compile from the sources, you will need:

  • ghostscript-5.50.tar.gz, Ghostscript sources 2826251 bytes
  • jpegsrc.v6b.tar.gz, JPEG library sources, 613261 bytes
  • libpng-1.0.2.tar.gz, LIBPNG library sources, 225386 bytes
  • zlib-1.1.3.tar.gz, ZLIB library sources, 168463 bytes
  • macgs-550-src.sit.bin, additional sources for the Macintosh, 227k
Start by reading the readme.txt file for installation instructions.

The MacGS Viewer Manual is part of the macgs-550-installer.bin archive and contains more detailed information. There is also a web version of the manual on-line. You can find out more about MacGS-550 on-line as well.
