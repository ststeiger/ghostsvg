AFPL Ghostscript 8.54

Artifex Software, Inc. and artofcode LLC are pleased to announce the release of AFPL Ghostscript 8.54. This is a maintenance release in the stable 8.5x series.

Major new features include:

  • The COMPILE_INITS build define now generates a compressed read-only filesystem which is linked into the executable and accessible from the interpreter as a new %rom% iodevice. This both improves installed footprint and allows using the same mechanism for embedding Resource files and fonts as well as postscript library and configuration files.
  • This release also supports the proprietary Luratech JBIG2 and JPEG 2000 libraries. Commercial customers with an appropriate license can use these both for image decode and to create PDF files using these compression formats with the pdfwrite device.
  • All source releases now include all the 3rd part library dependencies. It it no longer necessary to download libpng, libjpeg or zlib separately if they are not installed on your system.
  • In addition there are a large number of bug fixes and rendering improvements.

Source is available for download from the usual locations:

ftp://mirror.cs.wisc.edu/pub/mirrors/ghost/AFPL/gs854/
http://sourceforge.net/projects/ghostscript/
MS Windows binaries and other packages will appear as they become available.

The basic font set is distributed separately. We recommend:

Post-release news

After you get this version of Ghostscript, we suggest that you track news, updates, and fixes here or at the Ghostscript community website, ghostscript.com.

Mirror sites

The primary site for the Ghostscript source code is http://prdownloads.sourceforge.net/ghostscript/. The primary FTP site for the full Ghostscript release is ftp://mirror.cs.wisc.edu/pub/mirrors/ghost/. Ghostscript is also available from many mirror sites which may be closer to you and provide faster access. Ghostscript is also available on CD-R.

GSview

If using OS/2 or MS-Windows it is strongly recommended that you also get GSview, which provides a graphical interface to Ghostscript.

To unpack ZIP files, you will need an unzip program such as Info-ZIP unzip.
