Obtaining Aladdin Ghostscript 4.03

See also the topic Ghostscript Licensing.
The primary FTP site for Ghostscript is ftp://ftp.cs.wisc.edu. Ghostscript is also available from many mirror sites which may be closer to you and provide faster access.

Unix & VMS

You will need to obtain the following sources:
  • ghostscript-4.03.tar.gz, Ghostscript sources, 2166k
  • jpeg-6a.tar.gz, JPEG library sources, 527k
  • libpng-0.89c.tar.gz, LIBPNG library sources, 115k
  • zlib-1.0.2.tar.gz, ZLIB library sources, 83k
and also the following fonts:
  • ghostscript-fonts-std-4.03.tar.gz, Ghostscript standard fonts, 1409k
Start by reading the README file in ghostscript-4.03.tar.gz.

It is also worthwhile having the Ghostview previewer.

OS/2, MS-DOS & MS-Windows

You will need to obtain the following ZIP files:
  • gs403ini.zip, configuration, initialization, and example files, 678k
Plus one or more of:
  • gs403dos.zip, MS-DOS 32-bit, 780k
  • gs403os2.zip, OS/2 32-bit, 572k
  • gs403w32.zip, MS-Windows 32-bit (Win32s, Win95, WinNT), 472k
  • gs403win.zip, MS-Windows 16-bit (Win16), 459k
If you don't already have a full set of Type 1 fonts, you will also need:
  • gs403fn1.zip, Ghostscript standard fonts, 1143k
The documentation for Ghostscript is in the gs403ini.zip file. Look for README, new-user.txt, install.txt and use.txt.

If using OS/2 or MS-Windows it is recommended that you also get GSview, which provides a graphical interface to Ghostscript. GSview 2.4 contains an installation program which also installs Ghostscript 4.03. You need gsview24.zip, gs403ini.zip, gs403fn1.zip and either gs403w32.zip or gs403os2.zip. See the README.TXT file in gsview24.zip for more details.

Macintosh

Aladdin Ghostscript 4.03 is not yet available precompiled for the Mac. The following information applies to Aladdin Ghostscript 3.33.

You will need to obtain one of the following:

  • macgs-v1.0-68k.sit.hqx, application for 68020 or better machines, 442k
  • macgs-v1.0-ppc.sit.hqx, application for PowerPC version, 470k
plus all of these:
  • readme.txt, installation instructions, 2k
  • macgs-v1.0-files.sit.hqx, configuration, initialization, and example files, 719k
  • macgs-v1.0-fonts.sit.hqx, full set of Ghostscript's Type 1 fonts, 2,795k
If you plan to re-compile from the soruces, you will need:
  • ghostscript-3.33.tar.gz, Ghostscript sources
  • ghostscript-3.33jpeg.tar.gz, JPEG library sources
  • macgs-v1.0-src.sit.hqx, additional sources for the Macintosh, 186k
Start by reading the readme.txt file for installation instructions.

The Mac GS Viewer Manual is part of the macgs-v1.0-files.sit.hqx archive and contains more detailed information. There is also a web version of the manual on-line. You can find out more about Mac GS Viewer on-line as well.
