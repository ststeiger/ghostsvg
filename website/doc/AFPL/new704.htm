AFPL Ghostscript 7.04

Posted 1 Feb 2002 by raph

artofcode LLC and Artifex Software Inc. announce the release of AFPL Ghostscript version 7.04. This release is a security update and maintenance release in the 7.0x branch. In addition to the security fixes, we have also added an autoconf script and support for IJS printer drivers, both of which should make installing and configuring Ghostscript easier for most users.

As usual, the primary download sites are SourceForge and cs.wisc.edu.

Thanks to everyone who helped with this release.

Please see Obtaining AFPL Ghostscript 7.04 for details on how to get it.
