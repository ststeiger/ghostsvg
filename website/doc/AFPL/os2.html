OS/2 Ghostscript

With GSview, Ghostscript can be used as a PostScript file viewer for IBM WebExplorer.

Ghostscript can be used to provide Transparent PostScript printing with PrintMon.

Driver for HP DeskJet 850
This is not part of the standard distribution. Information about this driver is available from http://www.erdw.ethz.ch/~bonk/bonk.html. The sources are available from http://www.erdw.ethz.ch/~bonk/software.html
Send bug reports for this driver to uli@erdw.ethz.ch
A precompiled OS/2 EXE is containing this driver is available from http://www.erdw.ethz.ch/~bonk/software.html

Some detailed notes on using the PrintMon with the HP850 driver are available.
