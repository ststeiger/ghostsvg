gs4.01 and gs2.6.2 show different output when 
displaying a .ps file with the contents

%!PS-Adobe-2.0 
 newpath 300 100 moveto 
  80 100 80 300 30 arcto  4 {pop} repeat
stroke 
showpage

gs2.6.2 shows an arc with a straight line, 
gs4.01 shows only the arc but no line.

Changing line 173 in file gspath1.c in the gs4.01 distribution
from

   code = arc_add(pgs, true, arad, xt0, yt0, xt2, yt2, ax1, ay1);

to
   code = arc_add(pgs, arc_lineto, arad, xt0, yt0, xt2, yt2, ax1, ay1);

makes the recompiled gs4.01 show the same output 
as gs2.6.2 and a HP Laserjet 4.

Note that I am not at all sure if this fix has any negative side effects.
