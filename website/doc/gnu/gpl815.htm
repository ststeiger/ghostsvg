Release of GPL Ghostscript 8.15

artofcode, LLC. and Artifex Software, Inc. are pleased to announce the release of GPL Ghostscript 8.15. This is the latest stable release, and the first release of the 8.1x series under this license.

This release includes many bug fixes over the previous AFPL Ghostscript 8.14 release, improved font rendering, and offers significantly better PDF generation and handling over GPL 8.01. We recommend upgrading to all our free users.

Also new in this version is support for running Ghostscript in job server mode, so that multiple jobs can be piped to a single executable instance as if it were a printer. Just invoke gs with the new -dJOBSERVER switch and delimit jobs on stdin with ^D. See the Usage documentation for more details.

Source archives and binaries are available from the usual locations and mirrors:
