Release of GNU Ghostscript 7.06

We're pleased to announce the availablity of GNU Ghostscript 7.06, the second GPL release of the stable 7.0x branch. This release includes many bugfixes and improved conformance with GNU project guidelines.

The 7.06 release is available from sourceforge and wisc.edu as per usual.

Free Software users are encouraged to upgrade.

Thanks to everyone for their hard work in making this release possible
