Release of GNU Ghostscript 7.05

We're pleased to announce the availablity of GNU Ghostscript 7.05, the first GPL release of the stable 7.0x branch, 54 weeks after the first AFPL release.

The 7.05 release is available from sourceforge and wisc.edu as per usual.

The new release is also available from gnu.org and its mirrors.

New in this release is a port of the gs-cjk team's asian language improvements from the 6.5x branch, a major improvement over 7.04. PDF handling in general is much better than in the previous GNU release (6.53) and the portability of the autoconf build system is much improved. Of course there are the usual minor bugfixes as well. Free Software users are strongly encouraged to upgrade.

Thanks to everyone for their hard work in making this release possible
