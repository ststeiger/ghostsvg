Mirror Sites for Ghostscript

We have been told that you can find mirrors of

ftp://mirror.cs.wisc.edu/pub/mirrors/ghost/
on the following sites:
  • ftp://ftp.mirror.ac.uk/sites/ftp.cs.wisc.edu/ghost/ and http://www.mirror.ac.uk/sites/ftp.cs.wisc.edu/ghost/ (Canterbury and Lancaster, UK),
  • ftp://ftp.medasys-digital-systems.fr/pub/unix/ghostscript (Gif sur Yvette, near Paris, France),
  • ftp://ftp.fh-koblenz.de/pub/Ghostscript/ (Koblenz, Germany),
  • ftp://munnari.oz.au/ghost/ (Melbourne, Australia),
  • ftp://elza.lpi.ru/pub/ghost/ (Moscow, Russia).
  • ftp://sunsite.cnlab-switch.ch/mirror/ghost/ (Swiss SunSITE in Zurich, Switzerland)
  • ftp://ftp.kmitl.ac.th/pub/ghost/ (Bangkok, Thailand)
If any of this information is incorrect, or if you know of other mirror sites, please let us know.

The Comprehensive TeX Archive Network (CTAN) maintains a mirror of

ftp://ftp.cs.wisc.edu/ghost/
as
/support/ghostscript
on the following sites:
  • ftp://ftp.dante.de/tex-archive, (Mainz, Germany)
  • ftp://ctan.tug.org/tex-archive, (Boston, Massachusetts, USA)
  • ftp://www.tex.ac.uk/tex-archive, (Cambridge, UK)
Below is the list of CTAN mirror sites as of 1998-09-24. The most recent version of this list can be obtained from the file CTAN.sites at the above sites. Please use the CTAN mirror site or site nearest to you. 
In order to reduce network load, it is recommended that you use the
Comprehensive TeX Archive Network (CTAN) host which is located in the
closest network proximity to your site.  Alternatively, you may wish to
obtain a copy of the CTAN via CD-ROM (see help/CTAN.cdrom for details).
