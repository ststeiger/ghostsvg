Ghostscript API programming examples

Ghostscript is available as a DLL on Windows and a shared library on Linux. The documentation for the DLL/shared library API is in API.htm.

C / C++

The API documentation includes examples written in C. Other examples are the ghostscript source files src/dwmainc.c and src/dxmainc.c.

Delphi

There is an example available at ftp://mirror.cs.wisc.edu/pub/mirror/ghost/contrib/gsapi_delphi.zip
