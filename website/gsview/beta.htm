GSview BETA

The latest release version is GSview 4.3.

GSview 4.31 beta test is now available. This includes Windows, OS/2 and Linux executables. The major changes are:

  • Fixes to work with security updates in AFPL Ghostscript 8.00.
  • Added Catalan and Slovak languages. Help files are incomplete.
  • Use UTF-8 for Linux. If this isn't your default, you will need to do something like the following before running GSview.
    export LANG=en_AU.UTF-8
  • Several minor bug fixes.

GSview 4.31 beta can be found in ftp://mirror.cs.wisc.edu/pub/mirrors/ghost/ghostgum/beta/

Linux

A tar file containing a precompiled gsview executable is available in ftp://mirror.cs.wisc.edu/pub/mirrors/ghost/ghostgum/beta/gsview-4.31beta.i386.tar.gz.

RPM is available in ftp://mirror.cs.wisc.edu/pub/mirrors/ghost/ghostgum/beta/gsview-4.31-1.i386.rpm.

You will also need GNU Ghostscript 7.05 or AFPL Ghostscript 8.00 . In particular, you need to install both the ghostscript executable and shared library. These are available in ghostscript-8.00.i386.tar.gz.

Win32

GSview 4.31 beta for Win32 is available from ftp://mirror.cs.wisc.edu/pub/mirrors/ghost/ghostgum/beta/gsv431w32.exe.

You will also need Ghostscript 7.04, 8.00 or later.

OS/2

GSview 4.31 beta for OS/2 is available from ftp://mirror.cs.wisc.edu/pub/mirrors/ghost/ghostgum/beta/gsv431os2.zip

You will also need Ghostscript 7.04, 8.00 or later.

General

Sources are in ftp://mirror.cs.wisc.edu/pub/mirrors/ghost/ghostgum/beta/gsv431src.zip.

This beta will expire at the end of April 2003.
