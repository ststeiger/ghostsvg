GSview BETA

The latest release version is GSview 4.0.

GSview 4.05 beta test is now available. This includes Windows, OS/2 and Linux executables. The major changes are:

  • Greek translation
  • Dutch translation (except help file)
  • Much more portable on Unix. Compiles on Linux, SunOS, SGI IRIX and hopefully others.
  • New command line options for setting page size and orientation.
  • Progressive update of the display as Ghostscript renders each page.
  • New page sizes A0, A1 and A2 configured in the initialisation file.
  • Updated command line options.

GSview 4.05 beta can be found in ftp://mirror.cs.wisc.edu/pub/mirrors/ghost/ghostgum/beta/

Linux

A tar file containing a precomiled gsview executable is available in ftp://mirror.cs.wisc.edu/pub/mirrors/ghost/ghostgum/beta/gsview-4.05beta.linux.tar.gz.

You will also need AFPL Ghostscript 7.0. In particular, you will need to install the AFPL Ghostscript 7.00 RPM or tar.gz and the shared object tar.gz.

RPM is available in ftp://mirror.cs.wisc.edu/pub/mirrors/ghost/ghostgum/beta/gsview-4.05-1.i386.rpm.

Win32

GSview 4.05 beta for Win32 is available from ftp://mirror.cs.wisc.edu/pub/mirrors/ghost/ghostgum/beta/gsv405w32.exe.

You will also need AFPL Ghostscript 7.0.

OS/2

GSview 4.05 beta for OS/2 is available from ftp://mirror.cs.wisc.edu/pub/mirrors/ghost/ghostgum/beta/gsv405os2.zip

You will also need AFPL Ghostscript 7.0.

General

Sources are in ftp://mirror.cs.wisc.edu/pub/mirrors/ghost/ghostgum/beta/gsv405src.zip.
