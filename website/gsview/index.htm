GSview

GSview is a graphical interface for Ghostscript. Ghostscript is an interpreter for the PostScript page description language used by laser printers. For documents following the Adobe PostScript Document Structuring Conventions, GSview allows selected pages to be viewed or printed. GSview requires Aladdin Ghostscript 4.03-6.99. GSview is available for Win32/Win32s and OS/2. Older versions are available for Win16.

GSview was written by Russell Lang at Ghostgum Software Pty Ltd.
Ghostscript was written by Aladdin Enterprises.

See Obtaining GSview 3.4 and Obtaining Aladdin Ghostscript 6.01.

If using AFPL Ghostscript 6.50, with GSview 3.4, you must manually configure GSview with Options | Advanced Configure. Easy Configure will not work
