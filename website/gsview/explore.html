Using GSview as an IBM WebExplorer viewer

To configure WebExplorer to use GSview for viewing PostScript files, select
Configure | Viewers...
then select PS/EPS/AI PostScript from the Type drop down list box
then type gvpm.exe in the Program edit field. If GSview is not on your PATH, you will need to type the full pathname, e.g. c:\gs\gvpm.exe.

Make sure that the GSview "Options | Safer" is checked.

To test if your configuration is correct, display this PostScript file.

Updated on 9 Jan 1995 based on experience with IBM WebExplorer 1.0.
