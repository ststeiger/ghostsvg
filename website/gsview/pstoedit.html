GSview with PStoEdit

PStoEdit is a program by Wolfgang Glunz which uses Ghostscript to convert PostScript or PDF to an editable vector format.

PStoEdit has a DLL which can be used directly by GSview.

PStoEdit can be used from GSview 2.8, using "Edit | Convert to vector format".
For information on PStoEdit, see the PStoEdit home page at http://www.geocities.com/SiliconValley/Network/1958/pstoedit/
