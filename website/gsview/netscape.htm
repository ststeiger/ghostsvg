Using GSview as a Netscape Navigator Viewer

Netscape 4.05

To instruct Netscape Navigator for MS-Windows to use GSview as a viewer for PostScript files, use the following procedure: First install Ghostscript and GSview using the instructions in their documentation and verify that they are working correctly.

Start Netscape Navigator, then select "Edit", then "Preferences..."
In the "Category" list box select "Navigator" and "Applications".
Select "New Type", then enter

Description of type:PostScript file
File extension:ps eps
MIME Type:application/postscript
Application to use:c:\gstools\gsview\gsview32.exe

Press the "OK" button on the "New Type" dialog, then the "OK" button on the "Preferences" dialog.

Make sure that the GSview "Options | Safer" is checked.

Netscape 1.22

To instruct Netscape Navigator for MS-Windows to use GSview as a viewer for PostScript files, use the following procedure:

First install Ghostscript and GSview using the instructions in their documentation and verify that they are working correctly.

Start Netscape Navigator, then select "Options", then "Preferences...".
Select the "Helper Applications" tab.
From the Mime type list box select "application/postscript".
Select the "Launch Application" radio button, then the "Browse..." button.
From the Browse dialog box find and select the GSview or GSview32 EXE, then press the OK button.
Press the OK button on the "Preferences" dialog box.

Make sure that the GSview "Options | Safer" is checked.

To test if your configuration is correct, display this PostScript file.
