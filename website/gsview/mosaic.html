Using GSview as a NCSA Mosaic viewer

To instruct MS-Windows NCSA Mosaic to use GSview as a viewer for PostScript files, use the following procedure:

First install Ghostscript and GSview using the instructions in their documentation and verify that they are working correctly.

Edit mosaic.ini in your windows directory.
Find the section [Viewers]
In this section there should be a line
TYPE1="application/postscript"
A little further down should be the line
application/postscript="ghostview %ls"
Change this line to be
application/postscript="c:\gs\gsview.exe %ls"
replacing c:\gs\ with your path to GSview.
Save the modified copy of mosaic.ini.

Make sure that the GSview "Options | Safer" is checked.

To test if your configuration is correct, display this PostScript file.

Information about other viewers for NCSA Mosaic is available for MS-Windows and X11

This information was written on 2 November 1994.
